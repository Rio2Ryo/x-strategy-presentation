# 3. アルゴリズムを理解する（2分）

## スライド7: アルゴリズム攻略の3要素
1. **滞在時間** — ユーザーがポストに滞在する時間
2. **関心度** — 発信軸とグルーピングの一致
3. **親密度** — ユーザー間のやり取り頻度

## スライド8: エンゲージメントスコア（公開データ）
| アクション | スコア |
|-----------|--------|
| いいね | 0.5 |
| リポスト | 1.0 |
| リプライ | 13.5 |
| いいね/リポスト→プロフィールクリック | 12.0 |
| **もらったリプに投稿主がいいね/リプ** | **75.0** |
| ポストに2分以上滞在 | 10.0 |
| ミュート/ブロック | -74.0 |

**💡 ポイント: リプ返しでスコア75倍！**

## スライド9: 初動30分が勝負
- 投稿してすぐの反応で「拡散候補」と判断される
- ゴールデンタイム: 金曜12:00〜22:00
- 画像付き投稿は2倍のスコア
- Xプレミアムユーザーは4倍のスコア

## スライド10: UIハック手法（シロ調査）
1. **4枚画像トリミング** → タップ誘導で滞在時間UP
   - 例: マクドナルド（4枚画像クイズ）
2. **プロフ遷移誘導** → 「ヘッダーに収まりません」ミーム
   - 例: Google Japan（ヘッダー遷移）
3. **逆さ画像** → 物理的に巻き込む

# 4. アカウント設計の基本（2分）

## スライド11: 1アカウント1ジャンル
- 発信軸を明確にする → 正しくグルーピングされる
- 届けたい人に届きやすくなる
- フォロー率が高まる

## スライド12: HARMの法則
多くの人が関心を寄せる4ジャンル：
- **H**ealth（健康）
- **A**mbition（夢・キャリア）
- **R**elation（人間関係）
- **M**oney（お金）

## スライド13: プロフィール6要素
1. **アイコン** — 顔出しor印象的なイラスト
2. **ヘッダー** — 何者かが一目で分かる
3. **名前** — 覚えやすく、検索されやすい
4. **自己紹介** — 実績+提供価値+人柄
5. **固定ツイート** — 最も見せたいコンテンツ
6. **リンク** — 誘導先を1つに絞る

## スライド14: 日本特化パターン（シロ調査）
**「あきこちゃん方式」**
- キャラクター設定で親近感UP
- 企業感を消してユーザーが入り込みやすい
- 例: ローソン公式（大学2年生バイト設定）

| パターン | ニッチ | 具体例 |
|---------|--------|--------|

## スライド16: パターン①詳細 — AI活用×副業
- **ターゲット:** 会社員、副業初心者
- **発信内容:** AIツール紹介、実践レポート、収益公開
- **メリット:** 需要◎、ネタ尽きない
- **デメリット:** 競合多い、収益公開が必要
- **難易度:** ⭐⭐⭐☆☆

## スライド17: パターン②詳細 — スキル学習×初心者
- **ターゲット:** プログラミング/デザイン等の初心者
- **発信内容:** 学習記録、つまづきポイント、おすすめ教材
- **メリット:** 同じ境遇の人が共感、一緒に成長
- **デメリット:** 成長したら発信軸を変える必要
- **難易度:** ⭐⭐☆☆☆

## スライド18: パターン③詳細 — ニッチ特化×専門家
- **ターゲット:** 特定の悩み（HSP、ワーママ、地方移住）
- **発信内容:** 専門知識、体験談、解決策
- **メリット:** 競合少ない、濃いファンがつく
- **デメリット:** 市場が小さい、専門性が必要
- **難易度:** ⭐⭐⭐⭐☆

## スライド19: パターン④詳細 — キュレーション型
- **ターゲット:** 特定ジャンルの情報を追いたい人
- **発信内容:** ニュースまとめ、トレンド解説、考察
- **メリット:** 自分の意見少なくてOK、継続しやすい
- **デメリット:** 差別化が難しい、ファン化しにくい
- **難易度:** ⭐⭐☆☆☆

## スライド20: パターン⑤詳細 — ストーリー挑戦型
- **ターゲット:** 同じ目標を持つ人、応援したい人
- **発信内容:** 挑戦の過程、失敗談、学び
- **メリット:** 人間味、共感、応援したくなる
- **デメリット:** 成功しないと説得力がない
- **難易度:** ⭐⭐⭐☆☆

# 6. 成長フェーズ別戦略（2分）

## スライド21: フェーズ1（0→1,000フォロワー）
**期間目安:** 1〜3ヶ月

**やること:**
- 発信軸を1つに絞る
- 同ジャンルのアカウントに積極的にリプ
- 有益ポスト中心（7割以上）
- プロフィールを徹底的に作り込む

**目標指標:**
- 1日3〜5投稿
- 1日10〜20リプ
- フォロー率5%以上

## スライド22: フェーズ2（1,000→5,000フォロワー）
**期間目安:** 3〜6ヶ月

**やること:**
- バズを狙う「まとめ投稿」「賛否両論投稿」
- 同ジャンルの発信者と共同企画
- トレンドに自分の専門性を絡める
- 共感ポストを増やす（3割→5割）

**目標指標:**
- 1投稿あたり1万インプレッション
- エンゲージメント率3%以上

## スライド23: フェーズ3（5,000→10,000+フォロワー）
**期間目安:** 6ヶ月〜

**やること:**
- ファン化（共感ポスト中心）
- 収益化開始
- 外部メディア連携（note、YouTube等）
- コンテンツの横展開

**収益化条件:**
- Xプレミアム加入
- 過去3ヶ月で500万インプレッション
- プレミアムフォロワー500人以上
